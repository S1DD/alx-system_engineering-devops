<h1> ALX System Engineering Devops Track </h1>

<h3> Description </h3>
<p> At ALX Software school we are given projects on a weekly basis that we have to push to the git repository for an automatic review to be done by the checker.  This repository contains a list of files that correspond to a system engineering or dev ops project. We learn by doing, so not only are reading the course material but we are also applying what we learning to ensure understanding. In these projects we cover the basics of linux shell, bash commands etc. </p>
